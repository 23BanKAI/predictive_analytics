# Описание проекта

В рамках проекта реализована система virtual sensing для прогнозирования концентрации продукта процесса обжига на основе телеметрических данных.

Основная проблема заключается в том, что:

* лабораторные измерения поступают нерегулярно;
* измерения имеют задержку 10–15 минут;
* оператор не может получать мгновенную информацию о качестве продукта.

Для решения задачи были:

* выполнены разведочный анализ данных;
* синхронизация телеметрии и лабораторных измерений;
* инженерия временных признаков;
* построение и сравнение нескольких моделей машинного обучения;
* анализ остатков и интерпретация признаков.

# Интерпретация результатов

## MAE / RMSE

Показывают среднюю ошибку прогнозирования.

Чем меньше значения — тем лучше модель.

---

## MAPE / WAPE

Отражают относительную ошибку модели в процентах.

Используются для оценки практической точности virtual sensor.

---

## Direction Accuracy

Показывает способность модели правильно определять направление изменения процесса:

* рост концентрации;
* снижение концентрации.

---

## Анализ остатков

Проводятся статистические тесты:

* Jarque–Bera — проверка нормальности;
* Ljung–Box — проверка автокорреляции;
* Breusch–Pagan — проверка гетероскедастичности.

---

# Наиболее важные признаки

Наибольший вклад в прогноз обычно вносят:

* лаговые признаки;
* rolling statistics;
* EWMA признаки;
* признаки изменения режима работы печи.

Это подтверждает инерционность промышленного процесса.

# Лучший результат

Наилучшее качество показала модель:

```text
HistGradientBoostingRegressor
```

Причины:

* хорошо работает с tabular time-series;
* улавливает нелинейности;
* устойчив к шуму;
* эффективно использует engineered features.
