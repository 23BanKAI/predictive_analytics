# Лабораторная работа №3

# Классификация временных рядов и прогнозирование метеоданных шести городов России

## Описание проекта

В рамках лабораторной работы реализована двухэтапная система анализа временных рядов метеоданных.

Система выполняет:

1. Классификацию климатического режима по временному ряду метеоданных.
2. Выбор специализированной модели прогнозирования.
3. Прогноз температуры на 30 дней вперед.

Используемые города:

- Москва
- Санкт-Петербург
- Сочи
- Геленджик
- Благовещенск
- Находка

Период наблюдений:

```text
2019–2025
```

---

# Цель работы

Основные цели лабораторной работы:

1. Построить модель классификации климатических режимов.
2. Реализовать модели прогнозирования температуры.
3. Объединить оба этапа в единый pipeline.

# Преимущества двухэтапного подхода

1. Адаптация под различные климатические зоны.
2. Более высокая точность прогнозирования.
3. Интерпретируемость результатов.
4. Масштабируемость системы.

---

# Ограничения подхода

1. Ошибки классификации влияют на качество прогноза.
2. Качество прогнозирования ухудшается при горизонте более 30 дней.
3. Модели сглаживают экстремальные погодные аномалии.
